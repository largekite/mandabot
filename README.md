# MandaBot

> Learn Mandarin the fun way — built for little learners

MandaBot is an AI-powered Chinese language tutor that chats with kids in a friendly, encouraging way to build vocabulary, pronunciation, and confidence. Parents can track progress while kids enjoy bite-sized lessons through playful conversation.

Generated by **[Skylia](https://skylia.dev)** AI Startup Builder.

## Stack

| Layer | Service |
|-------|---------|
| Framework | Next.js 14 + TypeScript |
| Styling | Tailwind CSS + shadcn/ui |
| Hosting | Vercel |
| Database | Neon + Prisma |
| Auth | Clerk |
| Payments | Stripe |
| Email | Resend |
| AI | OpenAI |


## Getting started

```bash
npm install
cp .env.example .env.local
# Fill in your environment variables
npm run dev
```

### Where to get them

- `DATABASE_URL` — Neon PostgreSQL connection string (Neon dashboard)
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` — Clerk publishable key for frontend auth (Clerk dashboard)
- `CLERK_SECRET_KEY` — Clerk secret key for server-side auth (Clerk dashboard)
- `CLERK_WEBHOOK_SECRET` — Webhook signing secret for Clerk user sync events (Clerk dashboard)
- `OPENAI_API_KEY` — OpenAI API key for GPT-4o chat completions (OpenAI platform)
- `STRIPE_SECRET_KEY` — Stripe secret key for payment processing (Stripe dashboard)
- `STRIPE_WEBHOOK_SECRET` — Stripe webhook signing secret for subscription events (Stripe dashboard)
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` — Stripe publishable key for frontend checkout (Stripe dashboard)
- `RESEND_API_KEY` — Resend API key for sending weekly progress emails to parents (Resend dashboard)
- `NEXT_PUBLIC_POSTHOG_KEY` — PostHog project API key for analytics (PostHog dashboard)
- `NEXT_PUBLIC_POSTHOG_HOST` — PostHog ingest host URL (PostHog dashboard)
- `SENTRY_DSN` — Sentry DSN for error monitoring (Sentry dashboard)
- `NEXT_PUBLIC_APP_URL` — Public base URL of the deployed app (Vercel environment)
